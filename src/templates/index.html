NophiGene DRD4 Workbench

This workspace is built for red-biotech style analysis: variant review, methylation processing, and concise artifact generation around the DRD4 locus. The overview stays separate so the analysis surface can remain dense, wide, and practical once you start working with real data.

What this workbench expects

The app reads local project inputs from {{ data_dir }} and writes reports to {{ results_dir }}. It scans for VCFs, paired IDAT prefixes, and optional population statistics files so the browser stays close to the real project layout.

Variant layer: sequence calls from a regional VCF focused on DRD4.
Epigenetic layer: methylation beta values derived from paired Illumina IDAT files.
Output layer: a report artifact plus a companion methylation CSV for downstream review.

Why the workflow is split into tabs

The overview explains the biological workflow once. The analysis tab then uses nearly the full browser width and keeps the form and result panes visible together, so you do not have to keep scrolling past explanation blocks while entering or reviewing data.

Designed to keep the active work surface compact and high-density.
