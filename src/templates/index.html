Container-first genomics UI

NophiGene DRD4 Workbench

This interface is designed for the current project shape: mounted input folders, a Docker-run analysis pipeline, and generated artifacts written back into {{ results_dir }}. It keeps the browser layer thin and lets the existing Python workflow stay the source of truth.

What this UI expects

Mount your project data/ and results/ folders into the container. The app scans {{ data_dir }} for VCFs, IDAT sample prefixes, and optional population stats files.

Current output behavior

Each run produces a report file plus a companion methylation CSV. That keeps the UI simple while still giving you analysis artifacts you can inspect outside the browser.
